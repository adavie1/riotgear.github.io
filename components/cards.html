New in v3.6! The cards to display card content, with headers and optional footers.

Use

As with all RiotGear2 tags you can use markdown in your button text!

Card Attributes

These are global settings for rg-card tags.

style: <empty> | info | brand | warning | error | success

shadow: high | higher | highest

header attribute: [TEXT]

subhead attribute: [TEXT]

footer: [TEXT]


Footer Button Attributes (optional)

These are settings for each button in the footer section.

style: <empty> | info | brand | warning | error | success

active attribute: yes | no OR true | false | active  non-active

disbled attribute: yes | no OR true | false | disabled  enabled


HTML

The settings for header text, shadows & footer text can be set on a tag by tag basis. 

      <rg-card shadow="high | higher | highest" [header="TEXT"] [foooter="TEXT"]></rg-card>

JavaScript 


    var tags = riot.mount ("rg-card", {card: {
                            text: "My **precious** text, it's _mine_! It's _mine_!<br />",
                            header: {image: "https://unsplash.it/1024/500?random",
                                    text: "**Master** of the _Universe_",
                                    subhead: "Job Vacancy",
                                    divider: true,
//                                    style: "brand"
                                    },
                            footer: {
//                                    text: "_Footer_ Text",
//                                    divider: true,
//                                    style: "warning",
//                                    block: true,
                                    items: [
                                      {text: "Yes"},
                                      {text: "Maybe", style: "warning"},
                                      {text: "No", style: "error"}
                                    ]}
    }}) ;

    tags[0].on("button-clicked", function (e) {console.log("button-clicked:", e)}) ;
