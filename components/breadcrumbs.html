New in v3.6.0! The Blaze CSS 2 breadcrumb so you can create navigation and routes

Use

  • Useful if you want HTML text in your tags.
  • Actually this tag came about because of a need to "embed" HTML into the alerts

HTML 


	<rg-breadcrumb></rg-breadcrumb>

JavaScript 


     var tags = riot.mount('rg-breadcrumb', {
          breadcrumb: {
                  ...
                  }
        });
